## 🚀 快速开始

### 1. 安装
```bash
cd C:\msb-complete
pip install -e .
```

### 2. 设置API密钥
```bash
set OPENAI_API_KEY=your-key
set ANTHROPIC_API_KEY=your-key
set COHERE_API_KEY=your-key
```

### 3. 运行评估
```bash
# 基础评估
msb evaluate --model gpt-4 --dataset multilingual_safety --languages en,zh,es

# 使用Python API
python examples/basic_evaluation.py
```
